Technique for layering drum samples. (Mid/Side)

Introduction

I discovered this technique while I was trying to create a song only with Ableton’s bundled stuff. So I was forced into thinking of alternative ways to create cool sounding hihats.

That basically lead to this layout:

  • Dividing the hihat sound into 3 layers.
    1. One channel for the Mid signal
    2. Two Channels panned left/right for the sides signal

Controlling the drums ADSR (Attack, Decay, Sustain, Release) will be happeing into the samples itself. The goal is to create a similar an cohereant sounding signals. That resulted into a really nice souding hihat using only the stock samples.

I extened this and created a template layout for layering sounds. Which you can downloading below.

Layering Mid_Side.als

Breakdown into the layers:

Mid’s group has an EQ which is set in Mid/Side Mode, where I’ve deleted the sides signal. This channel will be playing only the mid signal.

Sides group is divided into 2 channels both of which are panneld 50L and 50R
